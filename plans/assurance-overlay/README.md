# Assurance Overlay — Companion Specs

Companion specification documents for the assurance capability, phased per
`PLAN-assurance-stpa-grc.md §24`. Each file covers one Phase's detailed
design artifacts (entity schemas, connection specs, attribute profiles, doc-types,
reference-vocab structure).

| File | Phase | Contents |
|---|---|---|
| `PHASE-0-module-extension.md` | 0 | Module extension mechanism decisions, `module_class` registry, attribute-profile convention |
| `PHASE-1b-entities.md` | 1b | Assurance ontology entity types (`entities.yaml` skeleton) |
| `PHASE-1b-connections.md` | 1b | Assurance connection types (`connections.yaml` skeleton) |
| `PHASE-1b-attribute-profiles.md` | 1b | `concern_class`, `disposition`, `uca_type`, TLP attribute schemas |
| `PHASE-1b-doc-types.md` | 1b | Assurance document-type specs |
| `PHASE-1c-reference-vocab.md` | 1c | Reference vocabulary structure (STRIDE/CWE/ATT&CK/obligation catalogs) |

Files are created as each phase begins.
