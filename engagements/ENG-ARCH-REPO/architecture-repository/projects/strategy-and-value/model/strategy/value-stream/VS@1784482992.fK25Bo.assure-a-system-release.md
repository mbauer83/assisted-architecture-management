## Assure a System Release

End-to-end value stream for safety/security analysts and risk & compliance officers: from an assurance obligation or concern for a release to a sealed, tamper-evident assurance case. Entrance: an assurance obligation or concern exists for a release. Exit: a sealed, tamper-evident assurance case exists for it. Stage value items: analysis scoped and anchored → identified losses/hazards/UCAs/constraints → external dependency risk bound to affected elements → risks dispositioned and obligations tracked → tamper-evident reviewable case.
