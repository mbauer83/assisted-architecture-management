## Grow Reusable Enterprise Knowledge

End-to-end value stream for architects and upper technical management: from engagement work that produces potentially reusable content to another engagement consuming the promoted knowledge — the two-tier product story. Entrance: engagement work produces potentially reusable content. Exit: another engagement consumes the promoted knowledge. Stage value items: candidate reusable knowledge → quality-assured promotion set → shared reviewed baseline grows → faster, consistent planning in other engagements.
