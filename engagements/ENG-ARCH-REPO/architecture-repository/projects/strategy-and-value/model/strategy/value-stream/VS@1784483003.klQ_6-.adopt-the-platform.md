## Adopt the Platform

End-to-end value stream for the prospective platform adopter: from learning the platform exists to a first engagement model that answers a real question. Entrance: a prospective adopter learns the platform exists. Exit: their first engagement model answers a real question for them. Stage value items: informed adoption decision → running, configured platform instance → guidance-assisted authoring enabled → first architecture model delivering answers. The most volatility-exposed of the four streams: wording is revisited at announcement time, while go-to-market content stays out of the model entirely.
