## Agent-Native Architecture Collaboration

The ability for humans and AI agents to co-author the same architecture knowledge with equal-fidelity surfaces: whatever one party can read, author, or verify, the other can too, at the same semantic depth. The ability itself is implementation-independent (enterprise at rest) — it would survive replacing every current interface. It is deliberately not a restatement of the principle "Equal-Capability Human and Agent Interfaces" (a standing design norm) nor of the requirement "Tool Interfaces: MCP, CLI, REST" (a testable obligation): the capability is the descriptive ability those norms and obligations exist to secure; the capability-to-requirement realization to "Tool Interfaces: MCP, CLI, REST" is the canonical bridge between the two.
