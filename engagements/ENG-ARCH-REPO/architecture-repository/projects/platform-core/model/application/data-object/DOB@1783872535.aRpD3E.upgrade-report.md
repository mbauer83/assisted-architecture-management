## Upgrade Report

Report of an upgrade run. Extended to multi-target operational upgrade: per target the kind, discovered and current versions, preflight, committed/rolled-back state, and the run's partial-apply/resume outcome — not only repository steps. Its accuracy after a mid-run failure is what makes a rerun safe: it names exactly which targets committed and which remain.
