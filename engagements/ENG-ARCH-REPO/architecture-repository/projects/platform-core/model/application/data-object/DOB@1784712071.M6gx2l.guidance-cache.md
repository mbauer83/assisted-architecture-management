## Guidance Cache

The deployed, deployment-level guidance cache (guidance-cache/ under the deployment, ~/.config/arch-repo/guidance-cache/ by default) — the imported, latest-format-only method guidance the running system reads. Materially distinct from the Guidance Import Source (the licensed source bundle): different location, lifecycle, and upgrade target. An older cache is migrated offline by the upgrade framework (a format-header patch, never re-fabricating content), and is read at runtime by Load Guidance Content.
