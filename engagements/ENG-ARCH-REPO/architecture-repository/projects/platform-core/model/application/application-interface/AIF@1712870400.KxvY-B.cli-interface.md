## CLI Interface

The command-line interface for human power users, scripting, and CI/CD pipeline integration. Exposes all repository operations as shell commands with structured arguments and output. Supports both interactive use and non-interactive (scriptable) mode with machine-parseable output formats (JSON, plain text).

## Properties

| Attribute | Value |
|---|---|
| Consumers | Power users, scripts, CI/CD pipelines |
| Output Formats | Human-readable, JSON |
| Protocol | CLI (stdin/stdout/stderr) |
