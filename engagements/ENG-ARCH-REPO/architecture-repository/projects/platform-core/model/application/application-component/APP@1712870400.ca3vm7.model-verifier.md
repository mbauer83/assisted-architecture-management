## Model Verifier

Application-layer verification orchestrator for entities, connections, documents, and diagrams. It depends on domain verification-contribution protocols and candidate-repository ports, invokes module-owned per-diagram and repository checks without importing datatype implementations, and delegates filesystem, indexing, PlantUML, scheduling, and persistence effects to injected ports and infrastructure adapters.
