## GUI Authoring Tool

Web-based SPA for browsing, searching, and authoring architecture and assurance content through the Architecture Backend REST interface. Architecture entities and diagrams expose an assurance lens; assurance findings navigate back to their model anchors and surface model-and-bind actions for modelling gaps.
