## Module Catalog

Immutable domain catalog of loaded ontology and diagram-type metadata, built once at startup and injected through RuntimeCatalogs. It exposes focused ontology, connection, diagram-type, specialization, and viewpoint catalog views including workspace identity prefixes and contribution hooks, while remaining independent of allocation, persistence, and verification effects.

| Module | src/domain/module_catalog.py |
