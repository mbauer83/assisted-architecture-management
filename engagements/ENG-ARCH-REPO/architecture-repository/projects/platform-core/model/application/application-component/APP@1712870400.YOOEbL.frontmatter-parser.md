## Frontmatter Parser

Parses YAML frontmatter from `.md` and `.puml` files. Extracts the structured metadata between `---` delimiters into a dictionary, handling YAML types (strings, lists, dates) and reporting parse errors. Used by both the verification system (to validate frontmatter) and the indexing system (to extract metadata for the SQLite index).

## Properties

| Attribute | Value |
|---|---|
| Input | File path or file content |
| Language | Python |
| Output | Parsed frontmatter dict + parse errors |
