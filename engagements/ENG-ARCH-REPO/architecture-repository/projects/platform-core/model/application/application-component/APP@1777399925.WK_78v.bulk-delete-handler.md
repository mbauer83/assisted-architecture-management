## Bulk Delete Handler

Orchestrates transactional execution of bulk delete operations, including preflight dependency validation, by running the staged bulk operation process within the MCP write server. Intentionally carries no direct requirement-realization connection: it is a write-pipeline internal composed into the Architecture Backend and governed by the backend's requirements.
