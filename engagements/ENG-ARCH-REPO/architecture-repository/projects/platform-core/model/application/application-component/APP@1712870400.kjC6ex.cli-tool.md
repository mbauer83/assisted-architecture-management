## CLI Tool

The command-line interface for repository operations. Provides commands for entity/connection/diagram creation, model verification, index management, querying, artifact promotion, deployment-level guidance import (arch-import-guidance), model exchange (arch-exchange), and version-aware repository format upgrade (arch-repair upgrade). Designed for power users, scripting, and CI/CD pipeline integration. Shares the same underlying service layer as the MCP server and GUI, ensuring consistent behavior across all interfaces.
