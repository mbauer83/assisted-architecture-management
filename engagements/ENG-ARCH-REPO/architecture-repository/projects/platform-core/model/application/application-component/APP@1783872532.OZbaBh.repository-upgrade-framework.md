## Repository Upgrade Framework

Hexagonal upgrade-step framework backing arch-repair upgrade: a step registry, evaluate/apply orchestration, and registered steps (e.g. the D9 multiplicity rename, an unrecognized-structure catch-all detector). Detects findings against a repo's format-contract version, applies auto-migratable ones atomically, and reports non-migratable ones with manual instructions — never silently skipped.
