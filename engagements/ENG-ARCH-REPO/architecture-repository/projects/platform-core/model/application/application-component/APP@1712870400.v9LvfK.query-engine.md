## Query Engine

Supports full-text search, metadata-based querying, graph-based relationship traversal, and viewpoint execution over the SQLite runtime index. Translates high-level query requests (from MCP, CLI, or REST interfaces) into SQL queries against the index tables, and a declared Viewpoint Definition's scope/selection into one of the execution representations (exploration, table, matrix, diagram). Supports TF-IDF ranking for full-text results, structured filtering/sorting for metadata queries, and recursive CTE-based traversal for graph queries.

## Properties

| Attribute | Value |
|---|---|
| Backend | SQLite FTS5 + recursive CTEs |

| Query Modes | full-text, metadata, graph-traversal |
