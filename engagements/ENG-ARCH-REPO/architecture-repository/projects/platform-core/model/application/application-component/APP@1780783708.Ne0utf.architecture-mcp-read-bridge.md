## Architecture MCP Read Bridge

Thin stdio↔HTTP client of the Architecture Backend providing read-only access to the architecture repository via MCP. Independently grantable trust boundary for read operations. Intentionally carries no direct requirement-realization connection: it is thin interface plumbing whose traceability is governed by the requirements realized by its endpoint adapter and its parent module.
