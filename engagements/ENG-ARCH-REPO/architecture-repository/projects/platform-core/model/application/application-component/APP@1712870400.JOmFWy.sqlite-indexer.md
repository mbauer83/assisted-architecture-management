## SQLite Indexer

Indexes repository-derived artifact metadata, workspace-identified diagram entities, and datatype attribute-type usages for efficient discovery and reverse-usage queries. The SQLite read model is disposable and refreshable from source files; it never authorizes or rejects repository writes, which are verified against candidate repository state.

## Properties

| Attribute | Value |
|---|---|
| Index Contents | entity metadata, connection graph, keywords, full-text |
| Language | Python |
| Output | .sqlite database file |
