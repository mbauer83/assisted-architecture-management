## Developer Workstation

The local developer machine where the architecture repository tools run. Hosts the Python runtime, PlantUML engine, git VCS, and the SQLite index database. All repository operations execute locally — there is no server component in the base architecture. The workstation accesses the git repository on the local filesystem and may push/pull to remote git origins for collaboration.

## Properties

| Attribute | Value |

| Key Software | Python 3.11+, PlantUML, Git, SQLite |
