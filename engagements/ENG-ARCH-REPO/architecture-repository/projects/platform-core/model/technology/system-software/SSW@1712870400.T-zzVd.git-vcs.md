## Git VCS

The Git version control system installed on the developer workstation. Provides repository initialization, commit history, branching, merging, and remote synchronization. Used directly by the tooling for git-history-based volatility analysis (via git log/blame) and implicitly by all repository operations that read/write files in the git working tree.

| Attribute | Value |

| Operations | commit, log, blame, push, pull, merge |
