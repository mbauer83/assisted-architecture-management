## Python Runtime

Python 3.13+ runtime that executes all repository tooling — the model verifier, write system, macro generator, SQLite indexer, query engine, MCP server, and CLI tool. Dependencies managed via uv. The Python runtime is the execution platform for all application-domain components. Code uses contemporary typing with PEP 604 union syntax (`X | Y`), PEP 585 built-in generics (`list[str]`, `dict[str, Any]`), and current PEP best practices.
