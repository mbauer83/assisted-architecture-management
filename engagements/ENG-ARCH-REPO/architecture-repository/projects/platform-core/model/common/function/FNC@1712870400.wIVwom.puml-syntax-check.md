## PlantUML Syntax Check

Validates PlantUML diagram source for syntactic correctness by invoking the PlantUML engine (plantuml.jar) in syntax-check mode. Parses the engine output to extract error locations and messages. This catches malformed markup, undefined macros, invalid relationship syntax, and structural errors before they result in broken rendered output.

## Properties

| Attribute | Value |
|---|---|
| Dependency | plantuml.jar on the host system |
| Input | PlantUML source file path |
| Output | Syntax check result (pass/fail with error locations) |
