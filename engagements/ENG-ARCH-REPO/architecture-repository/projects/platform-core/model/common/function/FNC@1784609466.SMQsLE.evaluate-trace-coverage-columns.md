## Evaluate Trace Coverage Columns

Branch-complete trace-pattern evaluation producing the row projection (a discriminated PatternResult union: authoritative verdict cells vs diagnostic observation cells) during viewpoint execution. Enumerates each row's branches over direct stored edges, tests leaf realization against the registry-derived eligible realizer set, and composes the row verdict from motivation completeness plus overall realization only.
