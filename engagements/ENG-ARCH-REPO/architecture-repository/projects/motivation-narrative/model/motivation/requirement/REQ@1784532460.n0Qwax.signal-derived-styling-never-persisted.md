## Signal-Derived Styling Never Persisted

Security-signal-derived styling and attributes are ephemeral and classification-bearing: they must never be written back into a git-tracked artifact. Renders that apply them are computed on demand and, where downloadable, leave only through the stamped export path — never by persisting the styled diagram into the model.
