## Entity-Level Security Posture Metrics

For an architecture entity, the system must expose security posture metrics — distinct open vulnerabilities, findings by directness and severity band, and maximum CVSS — computed from the active security refresh run, filtered to the viewer's classification ceiling and pinned to a single run snapshot so the figures are internally consistent.
