## Configurable Entities and Connections

Central aspects of entities and connections should be configurable: the frontmatter (one per file type) and the attribute profile (per specialization, one-to-one — an entity or connection carries at most one specialization, which carries its own attribute profile). Applies equally to entity and connection specializations.
