## Discovery Tools

The system shall provide tools for discovering and exploring architectural artifacts. This is a parent requirement grouping three specific discovery mechanisms: semantic full-text search, metadata-based querying, and graph-based relationship discovery. Together these ensure that both humans and AI agents can find relevant artifacts regardless of whether they know exact identifiers, metadata attributes, or relationship paths.
