## Motivation Coverage Reporting

The architecture must report branch-complete realization coverage across the motivation chain (goal to outcome to requirement), distinguishing a fully-realized branch from a merely existentially-reachable one so that an incomplete branch surfaces as an honest gap rather than a false green.
