## Semantic Full-Text Search

The system shall support semantic full-text search across all architectural artifacts — entities, connections, diagrams, and documents. Search shall index entity descriptions, notes, and frontmatter fields, enabling users and agents to find relevant artifacts using natural language queries. Implementation may use TF-IDF ranking or vector-based similarity, backed by the SQLite runtime index.
