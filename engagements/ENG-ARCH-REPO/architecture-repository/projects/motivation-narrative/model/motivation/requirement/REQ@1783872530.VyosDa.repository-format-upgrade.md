## Repository Format Upgrade

The system supports version-aware, idempotent migration of a repository's persisted format across software versions (arch-repair upgrade): dry-run reporting by default, a hard backend-not-serving gate before any write, auto-migratable findings applied atomically, non-migratable findings reported with manual instructions — never silently skipped.
