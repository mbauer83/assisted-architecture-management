## Assurance to Architecture Linkage

Every assurance analysis is anchored to the architecture element it assesses. Assurance nodes support one-way bindings to architecture entities; architecture views compute a confidential assurance lens and assurance views navigate back without persisting reverse references. Unbound analysis subjects are reported as modelling gaps with a model-and-bind workflow or a separation-of-duties task.
