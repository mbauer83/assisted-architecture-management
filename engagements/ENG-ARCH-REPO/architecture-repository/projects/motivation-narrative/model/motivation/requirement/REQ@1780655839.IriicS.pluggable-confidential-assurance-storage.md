## Pluggable, Confidential Assurance Storage

All assurance persistence — knowledge base, tamper-evident audit log/archive, and security-signals store — must be accessed via ports with swappable adapters so backend and confidentiality posture are per-deployment choices; encrypted at rest by default via SQLCipher; fail-closed gated; OS-keychain key; all signals TLP-tagged (default TLP:AMBER).
