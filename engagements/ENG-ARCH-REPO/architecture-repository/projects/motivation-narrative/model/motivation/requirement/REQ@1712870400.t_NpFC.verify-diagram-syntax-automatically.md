## Verify Diagram Syntax Automatically

The system shall automatically verify PlantUML diagram syntax when diagrams are created or modified. Syntax validation uses the PlantUML engine (plantuml.jar) to parse diagram source and report errors before they reach rendered output. This catches malformed markup, undefined macros, and structural errors early, preventing broken diagrams from entering the repository.
