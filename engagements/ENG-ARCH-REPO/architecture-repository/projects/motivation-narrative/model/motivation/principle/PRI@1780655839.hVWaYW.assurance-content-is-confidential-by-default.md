## Assurance Content Is Confidential by Default

All assurance artefacts, signals, and references are treated as confidential by default: encrypted at rest, TLP-tagged, accessed only through gated interfaces, with one-way architecture references that are never reverse-persisted.

## Properties

| Attribute | Value |
|---|---|
| Priority | Must |
| Rationale | Assurance artefacts contain attack-surface information and obligation-sensitive data; encrypting at rest, TLP-tagging, and gating access by default prevents inadvertent exposure to unauthorised consumers |
