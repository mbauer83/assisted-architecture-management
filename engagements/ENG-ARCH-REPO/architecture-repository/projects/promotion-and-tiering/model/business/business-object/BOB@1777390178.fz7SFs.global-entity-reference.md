## Global Entity Reference

The conceptual representation of a reference from an engagement repository to an entity promoted into (or living in) the enterprise repository. A reference carries EXACTLY the permitted connections of the entity type — and specialization — it references, resolved per instance at validation time; a type-level rule cannot express this, so the reference has no connection grammar of its own. One tier invariant restricts that surface: the global entity, and therefore its reference, is never the SOURCE of a directed relationship — the enterprise tier never depends on engagement content — so only incoming and symmetric relationships may attach to a reference.
