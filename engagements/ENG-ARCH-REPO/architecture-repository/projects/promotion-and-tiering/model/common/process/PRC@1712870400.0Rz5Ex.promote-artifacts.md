## Promote Artifacts

Promotes an explicit selected set of artifacts from an engagement repository into the enterprise repository, including conflict detection, validation, execution, replacement of promoted local artifacts with global references, and post-promotion verification. See the "Promote Artifacts" and "Promote Engagement Work to the Enterprise Repository" behavior views for step-by-step detail, and "One Enterprise Repository, Many Engagements" for the structural context this process operates within.

## Properties

| Attribute | Value |
|---|---|
| Output | Artifacts copied to enterprise repository + updated enterprise index |
| Trigger | Explicit request via promotion service |
