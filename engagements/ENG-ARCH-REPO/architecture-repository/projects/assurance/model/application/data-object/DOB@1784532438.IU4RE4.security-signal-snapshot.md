## Security Signal Snapshot

The persisted aggregate produced by one signal ingest: lifecycle state (staging, complete, active, superseded, or failed), canonical digest of the ingested bundle, and its component and vulnerability-finding rows. At most one snapshot is active per anchor; activating a new snapshot atomically supersedes the previous one. Posture computation reads from the active snapshot as the source of truth for security figures.
