## Security Refresh Run

The persisted aggregate for one signal refresh: its lifecycle state (staging, complete, active, superseded, or failed), the canonical digest of the ingested bundle, and its component, finding, and VEX rows. At most one run is active per anchor; activating a new run atomically supersedes the previous one. Every posture computation reads from the active run, making it the source of truth for security figures.
