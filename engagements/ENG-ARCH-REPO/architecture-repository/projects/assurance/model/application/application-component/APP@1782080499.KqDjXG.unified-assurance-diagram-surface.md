## Unified Assurance Diagram Surface

Frontend viewer surface that renders assurance diagram types (bowtie, control-structure, UCA-matrix) directly from the confidential assurance store, with selectable nodes and edges that open assurance detail panels. Replaces the generic diagram viewer for these types, which are no longer discoverable or renderable outside the assurance surface. GSN diagrams are dual-homed: they render in both the generic architecture viewer and the assurance bridge when confidential-store-backed.
