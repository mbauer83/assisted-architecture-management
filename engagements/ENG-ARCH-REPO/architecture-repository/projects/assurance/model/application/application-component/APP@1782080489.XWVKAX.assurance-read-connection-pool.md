## Assurance Read Connection Pool

Per-thread WAL connection manager for the confidential assurance store. Each thread in the anyio threadpool lazily obtains and caches its own SQLite connection opened in WAL journal mode with a busy-timeout. A generation counter invalidates all cached connections on lock/unlock cycles. Read operations from concurrent GUI sessions and agent threads execute fully in parallel without contention; only the WAL file is shared.
