## Resolve Assurance Edge Endpoints

Decorates policy-visible assurance edges with endpoint names and node types so navigation surfaces render links without client-side lookups. Confidentiality is upstream: only edges whose both endpoints pass the exposure policy are enriched — a lookup miss omits the edge entirely, never a placeholder. Serves the browse and node-detail surfaces through the Assurance REST Interface.
