## Compute Security Posture Metrics

Computes per-entity security posture metrics from the active Security Signal Snapshot: distinct open vulnerabilities, component findings by directness, severity-band counts, and maximum CVSS. Results are filtered to the caller's classification ceiling and pinned to one snapshot so a set of figures is always internally consistent. Performed by the Supply-Chain & Vulnerability Connector.
