## Provide Signal-Derived Viewpoint Attributes

Exposes the computed Security Posture Metrics as read-only, signal-sourced viewpoint attributes for styling and tabulation. These attributes are distinct from graph-derived attributes and are never written back into the model. Cooperates with the Query Binding Evaluator that evaluates viewpoint bindings and with the metric computation it draws its values from.
