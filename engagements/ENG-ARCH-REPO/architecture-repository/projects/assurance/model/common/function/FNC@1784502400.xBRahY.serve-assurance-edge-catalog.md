## Serve Assurance Edge Catalog

Serves the assurance edge-type catalog — connection types, the exhaustive per-pair permitted-relationship matrix, and the disjoint architecture-reference type catalog — straight from the loaded assurance ontology module. Reads module configuration, never store content, so it is available whenever the assurance capability is configured, without requiring the store to be unlocked.
