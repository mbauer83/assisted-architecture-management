## Delete Assurance Analysis

Use case that removes an assurance analysis from the confidential store. Blocks deletion if the analysis still contains nodes or edges (non-empty guard); otherwise removes the analysis record, emits an audit log entry, and returns a confirmed deletion status. Invocable via the Assurance MCP Endpoint Adapter (assurance_delete_analysis tool) and via the Assurance REST Interface (DELETE /api/assurance/analyses/:id). All writes run through the Assurance Write Queue.
