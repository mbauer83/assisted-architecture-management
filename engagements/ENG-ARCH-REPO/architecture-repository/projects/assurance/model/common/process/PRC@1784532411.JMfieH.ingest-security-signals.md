## Ingest Security Signals

Ingests external supply-chain and vulnerability signals — SBOM components, advisory version ranges, and VEX assessments — and produces a new Security Signal Snapshot for an architecture anchor. Reads the current Bill of Materials and Security Signals Store, stages and completes the snapshot, then activates it atomically while superseding the prior active snapshot. Emits Security Signals Ingested. Performed by the AI Agent role and associated with the DevOps Engineer as human operator.
