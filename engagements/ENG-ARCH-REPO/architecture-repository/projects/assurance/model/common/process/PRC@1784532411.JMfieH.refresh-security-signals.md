## Refresh Security Signals

Ingests external supply-chain and vulnerability signals — SBOM components, advisory version ranges, and VEX assessments — and produces a new security refresh run for an anchor. Reads the current Bill of Materials and the Security Signals Store, records a Security Refresh Run aggregate that stages, completes, and then activates (superseding the prior active run in one step), and emits the Security Signals Refreshed event. Run in the field through the refresh tooling; performed by the AI Agent role and, as the human operator, associated with the DevOps Engineer.
