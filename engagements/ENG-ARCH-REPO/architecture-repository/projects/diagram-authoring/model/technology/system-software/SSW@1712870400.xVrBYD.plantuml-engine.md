## PlantUML Engine

The PlantUML rendering engine (plantuml.jar) that validates diagram syntax and renders PUML source to image formats (PNG, SVG). Invoked by the PlantUML syntax check function for validation and by the diagram rendering pipeline for output generation. Requires a Java runtime on the host system.

## Properties

| Attribute | Value |
|---|---|
| Format | Java JAR (plantuml.jar) |
| Operations | Syntax check, render to PNG/SVG |
| Requires | Java Runtime Environment |
