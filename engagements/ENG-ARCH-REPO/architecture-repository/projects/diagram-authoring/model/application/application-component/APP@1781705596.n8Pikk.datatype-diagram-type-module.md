## Datatype Diagram Type Module

Pluggable datatype diagram module that owns classifier identity metadata, type-reference projection and resolution, E330–E336 verification contributions, discovery extraction, render preparation, and pure PlantUML rendering. It implements generic diagram-type extension protocols without introducing datatype dependencies into central orchestration.
