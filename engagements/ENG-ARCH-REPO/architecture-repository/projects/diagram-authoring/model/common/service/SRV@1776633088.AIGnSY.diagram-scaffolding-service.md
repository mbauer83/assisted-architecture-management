## Diagram Authoring Service

Provides diagram creation and editing through GUI, MCP, and CLI adapters. It allocates ontology-defined workspace identities before persistence, preserves immutable classifier IDs during edits, normalizes payload references atomically, and verifies candidate state before publishing.
