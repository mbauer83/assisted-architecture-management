## Datatype Diagram Type Module

Pluggable diagram-type module implementing the datatype (UML class) diagram. Comprises ontology.yaml (classifier entity type + dt-* connection types + allowed bindings), config.yaml (UI label, type_ui_slots, guidance), and renderer.py (restricted UML class-diagram PUML emitter). Loaded by the Module Catalog at startup. Source: src/diagram_types/datatype/.
