## Model Verifier

Core verification engine that validates entities, connections, and diagrams against model rules. Orchestrates frontmatter validation, referential integrity checks, PlantUML syntax checking, and status conformity enforcement. Supports full-model and incremental verification. Delegates rule logic to pure rule functions (no I/O inside rules); filesystem inventory, PlantUML execution, worker scheduling, and incremental state persistence are handled by injected application-owned ports (FileInventoryPort, PumlSyntaxPort, VerifierScheduler, IncrementalStatePort) with infrastructure adapters. Receives a RuntimeCatalogs instance at construction time — no module-level globals.
