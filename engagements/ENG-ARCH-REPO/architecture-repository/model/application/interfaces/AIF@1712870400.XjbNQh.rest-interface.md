## REST Interface

A future REST API interface for web-based integration, GUI backends, and third-party system access. Will expose discovery, querying, and read operations over HTTP with JSON payloads. Write operations may be restricted or require authentication. Shares the same underlying service layer as MCP and CLI interfaces.
