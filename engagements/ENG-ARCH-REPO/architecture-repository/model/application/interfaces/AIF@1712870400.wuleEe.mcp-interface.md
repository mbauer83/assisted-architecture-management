## MCP Interface

The Model Context Protocol interface through which AI agents access repository operations. Exposes authoring, validation, discovery, and querying tools as structured MCP tool definitions with typed input/output schemas. This is the primary integration point for AI coding agents running in Claude Code, Cursor, and similar MCP-aware development environments.

## Properties

| Attribute | Value |
|---|---|
| Protocol | MCP (Model Context Protocol) |
| Consumers | AI coding agents, AI planning agents |
| Operations | All authoring, validation, and discovery operations |
