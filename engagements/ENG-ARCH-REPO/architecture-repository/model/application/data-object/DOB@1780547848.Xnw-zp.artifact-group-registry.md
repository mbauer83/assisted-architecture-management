## Artifact Group Registry

Application-layer data for a group entry: stable opaque id, slug, display name, archived flag, and ordering hint — stored in .arch-repo/groups.yaml and held in-memory at runtime. Realizes the identity and lifecycle attributes of an Artifact Grouping that are not derivable from the directory path alone.
