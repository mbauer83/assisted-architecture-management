## Assurance REST Interface

HTTP REST interface that exposes assurance store operations to the GUI and any HTTP client. Provides endpoints for analysis lifecycle (list, create, delete), node and edge queries, statistics, coverage, risk register, BOM, vulnerability import, and diagram rendering. All write-path requests are routed through the Assurance Write Queue; read-path requests use the Assurance Read Connection Pool. TLP exposure policy is enforced before responses are returned.
