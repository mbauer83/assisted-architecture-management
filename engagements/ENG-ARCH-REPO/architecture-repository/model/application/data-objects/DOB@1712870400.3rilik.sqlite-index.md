## SQLite Index

The SQLite database file containing the runtime index of all repository metadata. Includes tables for entity metadata, connection metadata, full-text search (FTS5), keyword indices, and optionally git-derived volatility metrics. This is a derived, disposable artifact — always regenerable from the source files in the repository. Serves as the backend for all discovery, querying, and statistical operations.

## Properties

| Attribute | Value |
|---|---|
| Format | SQLite 3 database |
| Tables | entities, connections, fts_content, keywords, volatility |
| Lifecycle | Derived — regenerated from source on demand |
