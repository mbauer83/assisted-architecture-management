## Connection Metadata

The extracted connection data from .outgoing.md files, as stored in the SQLite runtime index. Each record represents one directed edge in the architectural model graph: source entity ID, target entity ID, connection type, and optional connection-specific metadata. These records enable graph-based traversal queries and dependency analysis.

## Properties

| Attribute | Value |
|---|---|
| Source | .outgoing.md file sections |
| Storage | SQLite index table |
| Fields | source-id, target-id, connection-type, description, file-path, section-offset |
