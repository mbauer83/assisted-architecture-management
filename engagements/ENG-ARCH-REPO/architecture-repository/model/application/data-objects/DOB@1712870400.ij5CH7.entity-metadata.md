## Entity Metadata

The extracted frontmatter metadata for a single entity, as stored in the SQLite runtime index. Contains all frontmatter fields (artifact-id, artifact-type, name, version, status, keywords, last-updated) plus derived fields (file path, domain, element-type from display block). This is the primary data structure for metadata-based querying and filtering.

## Properties

| Attribute | Value |
|---|---|
| Source | Entity file frontmatter |
| Storage | SQLite index table |
| Fields | All frontmatter fields + derived domain, element-type, file-path |
