## PlantUML Macro Library

The auto-generated `_macros.puml` file containing PlantUML macro definitions for all registered entities. Each macro defines an entity's alias, label, and ArchiMate stereotype, allowing diagrams to reference entities by short alias (e.g., `REQ_Cc2Dd2`) rather than embedding full element definitions. Regenerated by the macro generator whenever entities are created, modified, or deleted. Included in all diagram files via `!include`.

## Properties

| Attribute | Value |
|---|---|
| Format | PlantUML include file |
| Location | diagram-catalog/_macros.puml |
| Lifecycle | Derived — regenerated from entity §display blocks |
