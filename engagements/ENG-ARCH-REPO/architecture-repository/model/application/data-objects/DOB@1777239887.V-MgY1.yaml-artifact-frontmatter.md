## YAML Artifact Frontmatter

The YAML frontmatter section at the top of every artifact file (entities, connections, diagrams, documents). Delimited by --- markers. Contains structured metadata that is machine-parseable, indexable, and verifiable against configurable schemata. Core fields: artifact-id, artifact-type, name, version, status, keywords, last-updated. The frontmatter is the primary source of structured metadata for the runtime index.
