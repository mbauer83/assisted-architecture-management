## CLI Tool

The command-line interface for repository operations. Provides commands for entity/connection/diagram creation, model verification, index management, querying, and artifact promotion. Designed for power users, scripting, and CI/CD pipeline integration. Shares the same underlying service layer as the MCP server and GUI, ensuring consistent behavior across all interfaces.

| Language | Python |
| Framework | Typer or Click |
