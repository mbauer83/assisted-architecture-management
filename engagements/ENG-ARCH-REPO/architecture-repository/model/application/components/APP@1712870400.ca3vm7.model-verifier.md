## Model Verifier

The core verification engine that validates entities, connections, and diagrams against the model rules. Orchestrates frontmatter validation, referential integrity checks, PlantUML syntax checking, and status conformity enforcement. Supports both full-model verification (scan everything) and incremental verification (single artifact). Delegates to specialized functions (frontmatter-validation, referential-integrity-check, puml-syntax-check) and aggregates results into a structured verification report.
