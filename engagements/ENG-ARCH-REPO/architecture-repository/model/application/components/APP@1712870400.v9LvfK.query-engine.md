## Query Engine

Supports full-text search, metadata-based querying, and graph-based relationship traversal over the SQLite runtime index. Translates high-level query requests (from MCP, CLI, or REST interfaces) into SQL queries against the index tables. Supports TF-IDF ranking for full-text results, structured filtering/sorting for metadata queries, and recursive CTE-based traversal for graph queries (e.g., transitive closure of dependencies).

## Properties

| Attribute | Value |
|---|---|
| Language | Python |
| Query Modes | full-text, metadata, graph-traversal |
| Backend | SQLite FTS5 + recursive CTEs |
