## Macro Generator

Generates the `_macros.puml` library from entity `§display` blocks. Scans all entities in the `model/` directory, extracts their ArchiMate display metadata (alias, label, element-type), and produces PlantUML macro definitions that allow diagrams to reference entities by alias. Regenerated whenever entities are created, modified, or deleted, keeping the macro library synchronized with the model.

## Properties

| Attribute | Value |
|---|---|
| Language | Python |
| Module | src/tools/generate_macros.py |
| Output | diagram-catalog/_macros.puml |
