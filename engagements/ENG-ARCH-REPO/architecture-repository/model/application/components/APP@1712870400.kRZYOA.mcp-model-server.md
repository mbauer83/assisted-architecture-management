## MCP Model Server

The Model Context Protocol server that exposes repository operations as MCP tools for AI agent consumption. Wraps the authoring, validation, and discovery services as MCP tool definitions with structured input/output schemas. Enables AI coding agents to create entities, define connections, query the model, and verify integrity through the standard MCP protocol used by Claude Code, Cursor, and other AI development environments.
