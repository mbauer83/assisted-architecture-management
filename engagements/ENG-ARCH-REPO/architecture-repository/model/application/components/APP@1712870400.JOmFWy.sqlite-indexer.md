## SQLite Indexer

Indexes artifact metadata into a SQLite database for efficient querying. Extracts frontmatter fields, keywords, connection graph edges, and optionally git history metrics (volatility, change frequency). Supports full re-index and incremental updates. The SQLite index enables full-text search, metadata queries, and graph traversal that would be impractical over raw filesystem scans. The index is a derived, disposable artifact — always regenerable from source files.

## Properties

| Attribute | Value |
|---|---|
| Language | Python |
| Output | .sqlite database file |
| Index Contents | entity metadata, connection graph, keywords, full-text |
