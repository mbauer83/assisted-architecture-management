## Git Repository

The git repository containing all architecture files — the `model/` directory tree with entity and connection files, the `diagram-catalog/` with PUML diagrams and rendered output, and any document files. Git provides the persistence layer, version history, branching for experimental changes, and the collaboration mechanism (push/pull/merge). The repository is the authoritative source of truth; all other representations (SQLite index, rendered diagrams) are derived.
