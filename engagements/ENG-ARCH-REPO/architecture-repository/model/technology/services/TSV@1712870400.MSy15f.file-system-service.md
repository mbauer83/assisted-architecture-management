## File System Service

The local filesystem read/write service provided by the operating system. All entity files, connection files, diagram files, and configuration files are stored on the local filesystem. The tooling reads source files for parsing and verification, and writes files for entity/connection/diagram creation. The filesystem is the primary storage layer, with git providing versioning on top.

## Properties

| Attribute | Value |
|---|---|
| Provider | Operating system |
| Operations | read, write, list, stat, mkdir |
