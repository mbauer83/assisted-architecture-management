## Version Control Service

The version control service provided by the Git VCS. Exposes commit, log, blame, branch, merge, push, and pull operations. Consumed by the architecture tooling for volatility analysis (extracting change history) and by users/agents for collaboration (pushing/pulling changes). Provides the versioning, audit trail, and collaboration layer for the architecture repository.

## Properties

| Attribute | Value |
|---|---|
| Provider | Git VCS |
| Operations | commit, log, blame, branch, merge, push, pull |
