## Artifact Authoring

The ability to create new architectural artifacts with tool-mediated support — including frontmatter auto-population, display block generation, index updates, and inline validation. Authoring tools ensure that every artifact enters the repository in a complete, valid, and indexed state, whether created by a human through the GUI or by an AI agent through MCP/CLI.
