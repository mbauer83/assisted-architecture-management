## Artifact Discovery

The ability to find, explore, and query architectural artifacts through multiple discovery mechanisms: semantic full-text search, metadata-based querying, and graph-based relationship traversal. This capability is what makes the repository useful as a knowledge base rather than merely a storage system — it enables stakeholders to answer questions about the architecture without knowing exactly where to look.
