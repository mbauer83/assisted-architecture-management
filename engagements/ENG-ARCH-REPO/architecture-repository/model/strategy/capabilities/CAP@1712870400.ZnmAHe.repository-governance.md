## Repository Governance

The ability to manage the two-tiered repository structure, artifact lifecycle (draft → active → deprecated), and promotion of artifacts from engagement repositories to the enterprise repository. This capability encompasses status management, promotion with transitive dependency inclusion, and enterprise-level quality gates — enabling distributed architectural work while maintaining centralized coherence.
