## GUI Interface

The graphical user interface through which human actors interact with the architecture repository. Provides visual exploration of the model (browsing by domain, navigating connections, viewing rendered diagrams) and guided authoring (form-based entity creation, visual connection definition). The GUI is the primary interaction point for non-technical stakeholders (product owners, management) and for visual architectural work (diagram review, cross-domain navigation).

## Properties

| Attribute | Value |
|---|---|
| Target Users | Architect, Product Owner, Upper Technical Management |
| Capabilities Exposed | Discovery, authoring, visualization |
