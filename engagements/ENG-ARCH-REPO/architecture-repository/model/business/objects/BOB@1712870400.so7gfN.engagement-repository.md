## Engagement Repository

A per-project or per-engagement scoped architecture repository. Contains locally relevant artifacts: draft entities, project-specific models, internal iteration work, and engagement-specific documents. Provides a workspace for autonomous architectural work within a team or project scope. Artifacts proven valuable can be promoted to the enterprise repository via the promotion mechanism, including their transitive dependencies.

## Properties

| Attribute | Value |
|---|---|
| Lifecycle | Created per engagement, may be archived after completion |
| Promotion | Selected artifacts promotable to enterprise repository |
| Scope | Single project or engagement |
