## Enterprise Repository

The organization-wide architecture repository containing globally reusable artifacts. Holds cross-cutting guidelines, enterprise-wide architectural models, reusable building blocks at every ArchiMate domain, and curated decision records. Serves as the authoritative reference for organizational architectural knowledge. Populated through promotion from engagement repositories, with quality gates ensuring that only verified, coherent artifacts enter the enterprise scope.

## Properties

| Attribute | Value |
|---|---|
| Scope | Organization-wide |
| Content | Cross-cutting guidelines, enterprise models, reusable building blocks |
| Governance | Quality gates on promotion, enterprise-level configuration |
