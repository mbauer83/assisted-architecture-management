## Connection File

An `.outgoing.md` file representing all outgoing connections from a single entity. Adjacent to the entity file with the same base name. Contains frontmatter (source-entity, version, status, last-updated), a `<!-- §connections -->` delimiter, and H3 sections for each connection (`### {connection-type} → {target-entity-id}`). Each section may include optional YAML metadata and a human-readable description.

## Properties

| Attribute | Value |
|---|---|
| Format | Markdown with YAML frontmatter |
| Naming | Same as entity file + .outgoing.md suffix |
| Connection ID | {source}---{target}@@{type} |
