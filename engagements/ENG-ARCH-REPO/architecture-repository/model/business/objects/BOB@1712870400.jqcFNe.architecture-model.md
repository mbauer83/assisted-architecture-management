## Architecture Model

The complete architectural model spanning all ArchiMate NEXT domains — the aggregate of all entity files, connection files, and their relationships. The model is not a single file but the coherent whole formed by the `model/` directory tree and the connections expressed in `.outgoing.md` files. It is the primary artifact managed by the repository system, serving as the single source of truth for architectural knowledge.

## Properties

| Attribute | Value |
|---|---|
| Structure | model/{domain}/{type-plural}/ directory tree |
| Domains | motivation, strategy, common, business, application, technology |
| Connections | .outgoing.md files adjacent to entity files |
