## Diagram File

A PlantUML `.puml` file representing an architectural diagram. Contains YAML frontmatter (artifact-id, name, diagram-type, status), includes for the stereotype library and macro library, and PlantUML source that references entities via their PUML aliases. Stored in `diagram-catalog/diagrams/`. Rendered output goes to `diagram-catalog/rendered/`.
