## LLM-Based Agents Cannot Effectively Use Unstructured Architectural Knowledge

LLM-based coding agents and planning tools cannot reliably access or contribute to architectural knowledge stored in unstructured or visual-only formats; they require machine-readable, text-based, structured representations accessible through programmatic tool interfaces such as MCP or CLI.
