## Enable Cross-Engagement Architectural Reuse

Architectural patterns, decisions, standards, and element definitions proven within individual engagements should be promotable to the enterprise repository and adoptable across other engagements, reducing redundant architectural work and propagating validated design knowledge across the organization.

## Properties

| Attribute | Value |
|---|---|
| Measurability | Adoption rate of enterprise repository patterns in new engagements; reduction in re-creation of architecturally equivalent elements across engagements. |
| Priority | Must |
