## Provide Development Guidance for AI

Enable AI coding agents to retrieve relevant architectural context for spec-driven development. AI agents need to query the repository for component boundaries, interface specifications, technology constraints, and design rationale before generating code. The MCP and CLI tool interfaces provide structured access to entity metadata, connections, and diagrams, allowing agents to ground their code generation in architectural intent.

| Attribute | Value |
|---|---|
| Priority | Must |
| Measurability | AI agents can retrieve relevant architectural context through MCP/CLI before code generation |
