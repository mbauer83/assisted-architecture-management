## Assurance Diagram Outputs and Sources Are Classification-Gated

The diagram pipeline gates assurance diagram confidentiality by TLP classification. A publishable assurance diagram (TLP:WHITE or TLP:GREEN) renders to disk and persists its source to the shared diagram catalog like any diagram. A confidential assurance diagram (above the publishability ceiling, or unclassified) has its rendered output withheld from disk and its source redirected to a gitignored confidential location, so neither the image nor the .puml source reaches a shared repository.

## Properties

| Attribute | Value |
|---|---|
| Category | Security |
| Priority | Must |
| Rationale | Rule G-f already keeps confidential rendered images off disk; the same boundary must cover the diagram source, whose frontmatter and PlantUML body carry the same sensitive analysis content. Keying the gate to TLP classification — rather than a blanket per-type rule — lets a deliberately publishable assurance example ship in a public self-describing model while real confidential analyses stay protected by default. |
