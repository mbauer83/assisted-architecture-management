## Datatype (UML Class) Diagram Authoring

The system shall provide a datatype diagram: a UML class diagram restricted to data structure, supporting Class, DataType, Enumeration and Variant classifiers with typed attributes and UML structural relations (association, aggregation, composition, generalization). A classifier may be optionally bound to an ArchiMate data-object to trace fine-grained schema to the architectural data element it details.
