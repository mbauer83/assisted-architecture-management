## Grid-view for diagrams

A grid-based view of diagrams, where each grid-cell holds a card-view of a diagram with name, type, id, and if the type is ArchiMate a tag for the ArchiMate domain (or "Layered"). Each card should also have a button for downloading the diagram (as png or svg for ArchiMate and other PUML diagrams, as md or html for matrix-diagrams). This grid-view should be filterable by type and domain.
