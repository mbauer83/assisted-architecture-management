## Verification Tools

The system shall provide tools for verifying the correctness and consistency of the architectural model. This is a parent requirement grouping: verified referential integrity across model, diagrams, and documents; and automatic diagram syntax verification. Verification runs both at write time (inline validation) and as a full-model sweep (batch verification), ensuring the model never accumulates silent inconsistencies.
