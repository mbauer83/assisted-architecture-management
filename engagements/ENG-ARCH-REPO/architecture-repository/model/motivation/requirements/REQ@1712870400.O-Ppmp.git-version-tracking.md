## Git Version Tracking

The architecture repository shall be stored in a git repository, providing full version history for all artifacts. Git enables change tracking, blame analysis, branching for experimental architectural changes, and merge-based collaboration. The indexing system may also extract git history to calculate volatility metrics (absolute and relative change frequency), providing insight into which parts of the architecture are most actively evolving.
