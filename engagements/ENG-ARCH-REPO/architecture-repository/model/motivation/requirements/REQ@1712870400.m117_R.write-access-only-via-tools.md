## Write Access Only via Tools

All write operations to the architecture repository — creating entities, adding connections, producing diagrams — shall be performed exclusively through the provided tool interfaces (MCP, CLI, or GUI). Direct file editing bypasses frontmatter validation, index updates, referential integrity checks, and PUML macro regeneration. Tool-mediated writes ensure that every artifact enters the repository in a valid, complete, and indexed state.

## Properties

| Attribute | Value |
|---|---|
| Priority | Must |
| Category | Authoring (child of authoring-tools) |

## Notes

This is a design-time convention enforced by process, not a filesystem permission. The verification tools can detect artifacts that were created or modified outside the tool chain.
