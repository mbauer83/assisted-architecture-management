## PlantUML Diagrams

The system shall use PlantUML as the diagram format for all architectural visualizations. PlantUML source is text-based (git-trackable, LLM-producible), expressive (supports ArchiMate, UML sequence, activity, ER, and component diagrams), and renderable to multiple output formats. Diagrams reference entities via PUML macros generated from entity display blocks, ensuring that diagrams stay synchronized with the model.
