## Strictly Formatted Document Cross-References

All cross-references between documents (architectural decision records, guidelines, specification documents) and model entities shall follow a standardized format that is deterministically parseable. References shall use artifact IDs and support sub-anchors for section-level linking. This enables the verification system to check that all document cross-references resolve, and the indexing system to build a complete dependency graph that includes documents alongside model entities.
