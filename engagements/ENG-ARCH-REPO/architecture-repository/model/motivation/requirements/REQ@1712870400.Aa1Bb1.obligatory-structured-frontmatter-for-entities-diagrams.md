## Obligatory Structured Frontmatter for Entities and Diagrams

Every entity and diagram file shall contain YAML frontmatter conforming to a defined schema. For entities: artifact-id, artifact-type, name, version, status, keywords, and last-updated. For diagrams: artifact-id, artifact-type, name, version, status, keywords, diagram-type, and last-updated. Frontmatter is validated at write time by the authoring tools and at verification time by the model verifier. This structured metadata enables indexing, querying, and automated processing of all artifacts.
