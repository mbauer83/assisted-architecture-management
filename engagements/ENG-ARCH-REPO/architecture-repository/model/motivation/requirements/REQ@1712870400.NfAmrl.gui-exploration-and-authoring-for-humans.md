## GUI Exploration and Authoring for Humans

The system shall provide a graphical user interface that allows human users to explore the architectural model visually and author new artifacts through guided forms. The GUI shall support browsing entities by domain, navigating connections, viewing rendered diagrams, and creating new entities/connections with frontmatter auto-populated. This complements the CLI and MCP interfaces which serve AI agents and power users.
