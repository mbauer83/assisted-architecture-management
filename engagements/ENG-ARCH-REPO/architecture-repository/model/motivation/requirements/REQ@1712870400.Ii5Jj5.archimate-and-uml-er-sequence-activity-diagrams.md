## ArchiMate and UML/ER/Sequence/Activity Diagrams

The system shall support multiple diagram types beyond ArchiMate views: UML sequence diagrams (for interaction flows), UML activity diagrams (for process flows), ER diagrams (for data models), and UML component diagrams (for structural decomposition). All diagram types use PlantUML syntax and can reference model entities via the shared macro library, maintaining traceability between dynamic/structural views and the architectural model.
