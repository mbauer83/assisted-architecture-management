## Tool Interfaces: MCP, CLI, REST

The system shall expose its capabilities through multiple tool interfaces: MCP (Model Context Protocol) for AI agent integration, CLI for command-line automation and scripting, and REST API (future) for web-based integration. All interfaces share the same underlying service layer, ensuring consistent behavior regardless of access method. The MCP interface is the primary integration point for AI coding agents; the CLI serves power users and CI/CD pipelines; REST enables GUI backends and third-party integration.
