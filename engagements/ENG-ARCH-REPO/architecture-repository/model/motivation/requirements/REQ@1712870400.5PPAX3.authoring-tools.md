## Authoring Tools

The system shall provide tools for creating and modifying architectural artifacts. This is a parent requirement grouping: write-access-only-via-tools (all mutations go through validated tool interfaces) and GUI exploration and authoring for humans. Authoring tools enforce frontmatter schema compliance, generate default display blocks, update indices, and validate referential integrity at write time.
