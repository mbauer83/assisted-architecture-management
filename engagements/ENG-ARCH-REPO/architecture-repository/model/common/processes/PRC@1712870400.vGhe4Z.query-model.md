## Query Model

The process of searching and traversing the architectural model via the various discovery mechanisms. Accepts queries in three modes: full-text search (natural language against indexed content), metadata query (structured filters on frontmatter fields), and graph traversal (follow connections from a starting entity, with optional type/direction filters). Returns ranked results with artifact metadata and content previews.

## Properties

| Attribute | Value |
|---|---|
| Trigger | User/agent request via discovery service |
| Output | Ranked result set with metadata and content |
