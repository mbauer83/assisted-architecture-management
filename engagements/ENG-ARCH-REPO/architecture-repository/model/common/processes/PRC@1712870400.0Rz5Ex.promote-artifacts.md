## Promote Artifacts

Promotes an explicit selected set of artifacts from an engagement repository into the enterprise repository, including conflict detection, validation, execution, replacement of promoted local artifacts with global references, and post-promotion verification.

## Properties

| Attribute | Value |
|---|---|
| Output | Artifacts copied to enterprise repository + updated enterprise index |
| Trigger | Explicit request via promotion service |
