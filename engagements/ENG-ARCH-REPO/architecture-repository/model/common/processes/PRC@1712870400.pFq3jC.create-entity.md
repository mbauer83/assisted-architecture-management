## Create Entity

The process of creating a new model entity: generate a unique ID (epoch + random), determine the target directory from entity type and domain, populate frontmatter from the applicable schema, generate the default ArchiMate display block, write the entity file, create an empty .outgoing.md stub, update the PUML macro library, and update the runtime index. All steps are atomic — if any validation fails, no files are written.

## Properties

| Attribute | Value |
|---|---|
| Trigger | Author request via authoring service |
| Output | Entity .md file + .outgoing.md stub + updated index + updated macros |
