## Create Connection

The process of adding a connection between two entities: validate that both source and target entities exist, validate the connection type against ArchiMate NEXT relationship rules for the given element types, append a new H3 section to the source entity's .outgoing.md file (or create the file if it doesn't exist), and update the runtime index with the new connection. The connection ID is reconstructed as `{source}---{target}@@{type}`.

## Properties

| Attribute | Value |
|---|---|
| Trigger | Author request via authoring service |
| Output | Updated .outgoing.md file + updated index |
