## Indexing Service

Manages runtime indexing of architectural artifacts into SQLite for efficient querying. Extracts frontmatter metadata, connection graph structure, keywords, and optionally git history metrics (volatility, change frequency). Supports both full re-index (scan entire repository) and incremental index (update changed files). The index is a derived artifact — the filesystem remains authoritative, and the index can always be regenerated from source files.
