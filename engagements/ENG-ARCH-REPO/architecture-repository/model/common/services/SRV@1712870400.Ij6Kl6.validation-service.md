## Verification Service

Exposes model & diagram verification operations to consumers. Supports both incremental verification (single artifact at write time) and full-model verification (batch sweep across all entities, connections, diagrams, and documents). Delegates to the model verifier, frontmatter validator, referential integrity checker, and PlantUML syntax checker. Returns structured verification results indicating errors, warnings, and passed checks.

## Properties

| Attribute | Value |
|---|---|
| Interface | MCP, CLI |
| Operations | verify-model (full), verify-entity, verify-connection, verify-diagram |
