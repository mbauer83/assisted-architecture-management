## Repository Promotion Service

Exposes the artifact promotion operation that moves artifacts from an engagement repository to the enterprise repository. Handles transitive dependency calculation (all entities and connections that the promoted artifact depends on are included), ID uniqueness validation against the enterprise repository, and quality gate checks (status, completeness, referential integrity). Returns a promotion plan for review before execution.
