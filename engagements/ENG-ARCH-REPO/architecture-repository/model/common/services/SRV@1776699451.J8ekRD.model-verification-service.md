## Model Verification Service

Exposes model verification operations to consumers. Supports both incremental verification (single artifact at write time) and full-model verification (batch sweep across all entities and connections). Delegates to the model verifier, frontmatter validator, referential integrity checker. Returns structured verification results indicating errors, warnings, and passed checks.
