## Discovery, Querying & Stats Service

Exposes artifact discovery, querying, and statistical analysis operations. Supports semantic full-text search, metadata-based filtering and sorting, graph-based relationship traversal, and aggregate statistics (entity counts by domain/type/status, connection density, volatility metrics). All queries operate over the SQLite runtime index for performance, with the filesystem as the authoritative source for content retrieval.
