## Authoring Service

Exposes artifact creation and modification operations to consumers (authors — human or AI). Accepts requests to create entities, define connections, and produce diagrams, delegating to the underlying application components for frontmatter generation, validation, index updates, and macro regeneration. The service ensures that every write operation results in a complete, valid, and indexed artifact regardless of which interface (MCP, CLI, GUI) initiated the request.

## Properties

| Attribute | Value |
|---|---|
| Interface | MCP, CLI, GUI |
| Operations | create-entity, create-connection, create-diagram, update-entity, update-connection |
