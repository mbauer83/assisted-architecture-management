## Referential Integrity Check

Verifies that all references between artifacts resolve correctly. For connections: checks that source and target entity IDs in .outgoing.md files correspond to existing entity files. For diagrams: verifies that all entity aliases used in PlantUML source match registered entities. For documents: validates that cross-reference artifact IDs point to existing artifacts. Also enforces status conformity: draft entities may only be referenced in draft diagrams.

## Properties

| Attribute | Value |
|---|---|
| Input | Artifact references + entity/connection registry |
| Output | List of unresolved references and status violations |
