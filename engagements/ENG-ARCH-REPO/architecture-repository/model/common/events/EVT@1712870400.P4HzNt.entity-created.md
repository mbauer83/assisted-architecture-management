## Entity Created

An event emitted when a new entity is successfully created in the repository. Triggers downstream actions: PUML macro library regeneration, runtime index update, and optionally notification to connected systems. Carries the new entity's artifact-id, type, and domain as event payload.

## Properties

| Attribute | Value |
|---|---|
| Trigger | Successful completion of create-entity process |
| Payload | artifact-id, artifact-type, domain, file-path |
