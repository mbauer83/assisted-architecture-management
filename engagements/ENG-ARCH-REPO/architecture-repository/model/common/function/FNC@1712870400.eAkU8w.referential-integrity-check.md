## Referential Integrity Check

Verifies reference integrity against authoritative candidate repository state, including ordinary artifact references, workspace-identity uniqueness and format, datatype attribute-type resolution, scope/status conformity, and safe classifier removal. Derived indexes support discovery but do not determine acceptance.

## Properties

| Attribute | Value |
|---|---|
| Input | Artifact references + entity/connection registry |
| Output | List of unresolved references and status violations |
